How to prevent the PopupContainerControl from being hidden

To accomplish your task, you need to use a regular form with the TopMost option enabled. In this case, you can manually specify when the form should be hidden. For more information, please see the following Knowledge Base article: http://www.devexpress.com/scid=A439 This example demonstrates how you can show a custom form at a required location.
